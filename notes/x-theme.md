## Theme manager

The best theme manager so far is, `mate-appearance-properties`, which
can be installed using:

```bash
sudo apt install mate-control-center
```

Others are:
1. lxappearance
1. gtk-theme-config
1. gtk-chtheme
1. qtconfig-qt4
1. xfce4-appearance-settings
